# 阴极保护智能运维平台

## 功能介绍

这是一个完整的阴极保护智能运维平台前端应用，包含：

### 页面功能

#### index.html - 大屏首页
- 实时监控24个机柜状态
- 左侧曲线图显示参比电压趋势
- 中间机柜状态网格
- 右侧报警信息面板

#### 1.html - 机柜列表管理
- 完整的机柜数据表格
- 筛选、查询、排序、分页
- 更多菜单（编辑、报警设置、数据图、数据修改、删除）
- 右侧滑出面板编辑
- 数据图大屏（80vw宽）
  - 参比/电压/电流切换
  - 时间查询
  - 数据导出（CSV）
  - 6条曲线图 + 数据表格

## 技术栈

- HTML5 + CSS3 + JavaScript（无框架依赖）
- ECharts 5.4.3 - 数据可视化
- 响应式设计

## 使用方式

直接在浏览器中打开 HTML 文件即可使用，无需任何服务器配置。

## 文件说明

- `index.html` - 大屏首页
- `1.html` - 机柜列表管理页面
